但是历史记录是异步加载的，而且可能较大。是否应该在方向键按下时即时加载？这可能有延迟。

我们可以实现：如果检测到当前视频为单集（parts为空且page列表为空或size<=1），则进行历史记录切换。切换逻辑：

    获取当前bvid。

    从历史记录api中获取最近的播放记录。

    找到当前bvid在列表中的位置。

    播放上一个或下一个。

但这样做每次按键都会请求历史记录，效率低。可以预先加载一个小的历史记录快照（比如最近的10个），但仍有延迟。更好的做法是：当用户进入播放器时，如果检测到没有多集，就预加载历史记录的前N项作为“播放列表”，并设置当前索引。

但这是比较重量级的改动。也许需求是：如果当前视频是单个视频，就把它当做一个只有一项的播放列表，然后左右键无法切换，按了没反应。但原描述是“切换到历史视频的上一个/下一个”，所以需要实现。

我们可以在PlayerActivity中添加一个全局的历史播放列表缓存，在startPlayback时如果检测到当前视频没有分集且没有播放列表，就异步加载最近N条历史记录，并按照bvid找到索引，然后设置一个临时的历史播放列表。这个临时的播放列表可以用于方向键切换。

或者更简单地：每次按键时实时加载历史记录。因为按键不频繁，可以接受几百毫秒的延迟。我们可以使用现有的historyCursor接口，从0开始加载，但可能需要加载多页才能找到当前视频，不现实。

更好的方案：调用BiliApi.historyCursor，一次最多加载24条，可以请求max=0获取最新。然后查找当前bvid是否在返回的24条中，如果不在，则无法定位，放弃切换。

private fun loadHistorySwitchListIfNeeded() {
    if (historySwitchItems.isNotEmpty() && historySwitchIndex in historySwitchItems.indices) return
    if (historySwitchLoadingJob?.isActive == true) return
    historySwitchLoadingJob = lifecycleScope.launch {
        try {
            val history = withContext(Dispatchers.IO) {
                BiliApi.historyCursor(max = 0, ps = 24)
            }
            val list = history.items.map { card ->
                PlayerPlaylistItem(
                    bvid = card.bvid,
                    cid = card.cid,
                    aid = card.aid,
                    title = card.title
                )
            }
            val idx = list.indexOfFirst { it.bvid == currentBvid }
            if (idx >= 0) {
                historySwitchItems = list
                historySwitchIndex = idx
            }
        } catch (_: Exception) {
            // 静默失败，不影响正常播放
        }
    }
}

3. 添加通过历史列表切换视频的方法

private fun playHistoryAdjacent(direction: Int) {
    val list = historySwitchItems
    if (list.isEmpty() || historySwitchIndex !in list.indices) return
    val targetIndex = historySwitchIndex + direction
    if (targetIndex !in list.indices) return
    val item = list[targetIndex]
    historySwitchIndex = targetIndex
    startPlayback(
        bvid = item.bvid,
        cidExtra = item.cid,
        epIdExtra = item.epId,
        aidExtra = item.aid,
        initialTitle = item.title,
        startedFromList = PlayerVideoListKind.PAGE // 或其他
    )
    showSeekHint(if (direction > 0) "下一个" else "上一个", hold = false)
}
